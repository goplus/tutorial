接口值可以使用 == 和 != 进行比较。两个接口值相等的条件是：两者都为 nil，或者它们的动态类型相同且动态值根据该类型的 == 的通常行为相等。因为接口值是可比较的，所以可以用作 map 的键或 switch 语句的操作数。
