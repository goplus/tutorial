在 XGo 中，惯用的做法是通过一个显式的、独立的返回值来传达错误。这与 Java 和 Ruby 等语言使用异常的方式不同，也不同于 C 语言中有时使用的重载单一结果/错误值的做法。XGo 的方式使得很容易看出哪些函数会返回错误，并使用与处理其他非错误任务相同的语言结构来处理它们。
---
