### 结构体的比较和转换
结构体是否可比较取决于其字段。完全由可比较类型组成的结构体是可比较的；包含切片或 map 字段的结构体则不可比较（正如我们将在后续章节中看到的，函数和通道字段也会使结构体不可比较）。

与 Python 或 Ruby 不同，没有可以重写的魔法方法来重新定义相等性，使 == 和 != 适用于不可比较的结构体。当然，你可以编写自己的函数来比较结构体。
